ES6 bundle

import $ from 'jquery';
import base from 'base/es6';
$(() => console.log(base.x + 42));

New JS standard. Until natively supported, ES6 modules can be transpiled to ES5 & CommonJS and then bundled like in the previous example.

  • + Script relations are explicit
  • + Tree requirements model
  • + Easy to have multiple versions, no version conflicts
  • + Easy to use with ES6
  • - Slower in development (bundle re-build on each change)
  • - Not perfectly integrated with Django
  • - Maybe harder to debug

Test it: open your browser console and look for messages. Also, take a look at the code in main/es6.html template and in the linked JS files.
