CommonJS bundle

var $ = require('jquery');
var base = require('base/common');

$(function() {
  console.log(base.x + 42);
});

CommonJS is default module loader in Node.js. It can be used to compile JS bundle on the server-side.

  • + Script relations are explicit
  • + Tree requirements model
  • + Easy to have multiple versions, no version conflicts
  • + Easy to use with ES6
  • - Slower in development (bundle re-build on each change)
  • - Not perfectly integrated with Django
  • - Maybe harder to debug

Test it: open your browser console and look for messages. Also, take a look at the code in main/common.html template and in the linked JS files.
