CommonJS

Server-side dependencies compilation. CommonJS is implemented in Node.js as default module loader (e.g. var $ = require('jquery')).

  • + Script relations are explicit
  • + Tree requirements model
  • + Easy to have multiple versions, no version conflicts
  • + Easy to use with ES6
  • - Slower in development (bundle re-build on each change)
  • - Not perfectly integrated with Django
  • - Maybe harder to debug

Test it: open your browser console and look for messages.
Also, take a look at the code in main/common.html template.
