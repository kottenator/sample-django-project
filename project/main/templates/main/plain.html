Plain <script> tags inclusion.

  • + Simple & most common option
  • + Fastest in development (no processing)
  • + Easy to debug
  • + Integrated with Django
  • - Script relations are implicit
  • - Maybe located in multiple files
  • - Maybe hard to have multiple versions or handle version conflicts
  • - Maybe hard to use with ES6
